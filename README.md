Haramaya University – Students Cafeteria Management System
This project is a web application built for Haramaya University to streamline cafeteria operations. It provides a secure login system where students can sign in using their registered first name and student ID to access cafeteria services. Administrators can manage student records and oversee cafeteria operations through the same platform.

✨ Key Features
Student Login: Access using first name and student ID.

Admin Access: Manage student data and cafeteria records.

User-Friendly Interface: Simple and responsive design for ease of use.

Theme Toggle: Option to switch between light and dark modes.

Secure Authentication: Ensures only registered students can access services.

🛠️ Tech Stack (if applicable)
Frontend: Likely React/Next.js (deployed on Vercel).

Backend: API integration for authentication and data management.

Deployment: Hosted on Vercel for fast and scalable access.

📍 Usage
Navigate to haramaya.vercel.app.

Enter your first name and student ID.

Sign in as a Student or contact the Admin for support.
